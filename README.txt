Features implemented

*WASD character movement
*Space to jump
*Score manager
*Finish zone
*Start screen
*Pause menu
*Animations for the character
*Moving Platform
*Activation button
*3D Audio
*Go to start menu when finished

Proposed improvements 

* Add different levels and use a different pick up mechanic in each one
* Add enemies that the player has to defeat to obtain stars
* Add life state and items to the character
* Add controller support and use vibration to add more feedback to the player
* Add post-processing to the scenes
